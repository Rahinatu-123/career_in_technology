Write a simple function that calculates the factorial of a number. This is a common programming task that demonstrates your problem-solving skills.
Text Input
function factorial(n) {
  if (n === 0 || n === 1) {
    return 1;
  }
  return n * factorial(n - 1);
}
// Test your function
console.log(factorial(5)); // Should output 120
